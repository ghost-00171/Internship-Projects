# Week 01 - Internship Notes

## Topic Studied
### OSI (Open Systems Interconnection) 7-Layer Model

During the first week of my Network Security internship, I studied the OSI 7-Layer Model to understand how data travels through a network. This model provides a structured way to understand network communication and troubleshoot connectivity issues.

## Layers Covered

1. **Physical Layer (Layer 1)**
   - Responsible for transmitting raw bits through cables and networking devices.
   - Learned about cables, connectors, hubs, and signal transmission.

2. **Data Link Layer (Layer 2)**
   - Handles communication between devices on the same network.
   - Studied MAC addresses, switches, Ethernet frames, and error detection.

3. **Network Layer (Layer 3)**
   - Responsible for routing packets between different networks.
   - Learned about IP addresses, routers, and packet forwarding.

4. **Transport Layer (Layer 4)**
   - Ensures reliable data delivery.
   - Studied TCP and UDP protocols, ports, and flow control.

5. **Session Layer (Layer 5)**
   - Establishes, manages, and terminates communication sessions between devices.

6. **Presentation Layer (Layer 6)**
   - Handles data formatting, encryption, decryption, and compression.

7. **Application Layer (Layer 7)**
   - Provides network services directly to end-user applications.
   - Learned about protocols such as HTTP, HTTPS, FTP, DNS, and SMTP.

## Key Learning Outcomes

- Understood the purpose of each OSI layer.
- Learned how data is encapsulated and de-encapsulated across the layers.
- Identified common protocols and devices associated with each layer.
- Built a strong foundation for networking and network security concepts.

## Skills Gained

- Basic networking concepts
- Understanding of OSI architecture
- Network troubleshooting fundamentals
- Knowledge of common network protocols
