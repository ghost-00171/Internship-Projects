# Internship Weekly Progress

## Week 1 – Networking Fundamentals

### Objectives
- Study the OSI 7-Layer Model.
- Understand how data travels across a network.
- Learn how web pages are retrieved from the internet.

### Activities
- Studied the functions of all seven OSI layers.
- Explored the process of retrieving a web page using DNS, TCP, HTTP/HTTPS, and IP.
- Learned how data is encapsulated and decapsulated as it passes through each OSI layer.
- Analyzed the communication flow between a client and a web server.

### Outcome
Developed a solid understanding of network communication and the role of each OSI layer in delivering web content.

---

## Week 2 – Network Design

### Objectives
- Design and configure a basic network topology.
- Understand IP addressing and device connectivity.

### Activities
- Designed a small enterprise network using Cisco Packet Tracer.
- Configured routers, switches, and end devices.
- Assigned IPv4 addresses and verified end-to-end connectivity.
- Tested communication using ICMP (ping) and basic troubleshooting techniques.

### Outcome
Successfully designed and implemented a functional network while gaining hands-on experience with basic network configuration.

---

## Week 3 – Layer 2 Switching Technologies

### Objectives
- Study Layer 2 networking concepts.
- Understand switching technologies used in enterprise networks.

### Activities
- Studied MAC Address Tables and frame forwarding.
- Learned the Address Resolution Protocol (ARP).
- Explored the Spanning Tree Protocol (STP) to prevent switching loops.
- Configured Virtual LANs (VLANs).
- Implemented trunk links between switches.
- Studied Router-on-a-Stick inter-VLAN routing.
- Configured basic Port Security.
- Learned network segmentation techniques using VLANs.

### Outcome
Acquired practical knowledge of Layer 2 technologies and their role in building secure and scalable enterprise networks.

---

## Week 4 – Three-Tier Enterprise Network Design

### Objectives
- Study and implement a hierarchical enterprise network architecture.

### Activities
- Designed a three-tier enterprise network in Cisco Packet Tracer.
- Implemented the Core Layer for high-speed backbone connectivity.
- Configured the Distribution Layer for routing, policy enforcement, and inter-VLAN communication.
- Configured the Access Layer for end-device connectivity.
- Verified communication between different VLANs and network segments.
- Tested the complete topology and performed basic troubleshooting.

### Outcome
Successfully implemented a hierarchical enterprise network following Cisco's three-tier architecture and gained practical experience in enterprise network design.
