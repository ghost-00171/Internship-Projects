# Week 3 – Layer 2 Switching Technologies

## Overview

During the third week of my internship, I focused on Layer 2 networking concepts and switching technologies used in enterprise networks. The objective was to understand how switches forward traffic, segment networks, and improve network performance and security.

## Topics Covered

### MAC Address Table
- Learned how switches build and maintain MAC address tables.
- Understood frame forwarding, filtering, and flooding processes.
- Examined how switches dynamically learn source MAC addresses.

### Address Resolution Protocol (ARP)
- Studied the purpose of ARP in mapping IPv4 addresses to MAC addresses.
- Observed the ARP request and ARP reply process.
- Analyzed ARP communication within the same broadcast domain.

### Spanning Tree Protocol (STP)
- Learned the importance of STP in preventing Layer 2 switching loops.
- Understood the root bridge election process.
- Studied port roles and port states in STP.

### Virtual LANs (VLANs)
- Configured multiple VLANs to logically segment the network.
- Learned how VLANs reduce broadcast traffic and improve security.
- Verified communication between devices within the same VLAN.

### Trunk Links
- Configured trunk ports between switches.
- Studied IEEE 802.1Q tagging.
- Allowed multiple VLANs to traverse a single physical link.

### Router-on-a-Stick
- Implemented inter-VLAN routing using a single router interface.
- Configured router subinterfaces with 802.1Q encapsulation.
- Verified communication between different VLANs.

### Port Security
- Configured basic port security on switch interfaces.
- Restricted unauthorized devices from accessing the network.
- Studied secure MAC address learning.

### Network Segmentation
- Learned how VLANs create separate broadcast domains.
- Understood the advantages of logical segmentation in enterprise networks.
- Improved network performance and security through proper segmentation.

## Practical Implementation

Using Cisco Packet Tracer, I implemented:
- VLAN creation and assignment
- Trunk configuration
- Router-on-a-Stick
- Basic Port Security
- End-to-end connectivity testing
- Layer 2 troubleshooting

## Skills Acquired

- Layer 2 Switching
- VLAN Configuration
- Trunking (802.1Q)
- Inter-VLAN Routing
- Port Security
- ARP Operation
- STP Fundamentals
- Network Segmentation
- Cisco Packet Tracer

## Outcome

By the end of Week 3, I gained practical experience with Layer 2 technologies and developed a strong understanding of how enterprise switches manage traffic, implement network segmentation, and provide secure communication within modern networks.
