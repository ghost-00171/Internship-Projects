# Week 02 - Internship Notes

## Topic Studied
### Small Network Design

During the second week of my Network Security internship, I learned how to design and configure a small network. The focus was on understanding how different networking devices communicate and how data flows within a local area network (LAN).

## Network Components Used

- Router
- Switch
- PCs/End Devices
- Ethernet Cables

## Concepts Covered

- Basic network topology
- IP addressing
- Subnet mask configuration
- Default gateway
- Device connectivity
- Router and switch roles
- End-to-end communication

## Practical Work

- Created a small network topology.
- Assigned IP addresses to all devices.
- Configured the default gateway.
- Connected devices through a switch.
- Verified communication using the **ping** command.
- Fixed basic connectivity issues during testing.

## Key Learning Outcomes

- Understood the difference between a router and a switch.
- Learned how to assign IP addresses correctly.
- Configured a basic LAN.
- Verified network connectivity between devices.
- Improved troubleshooting skills for simple network problems.

## Skills Gained

- Basic network design
- IP addressing
- Router fundamentals
- Switch fundamentals
- Network troubleshooting
- Connectivity testing using Ping
