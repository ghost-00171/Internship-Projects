# Week 05 - Enterprise Network Design Using Static Routing

## Overview

During the fifth week of my Network Security internship at Cyberster, I designed and implemented a complete enterprise-level network using Cisco Packet Tracer. The network was built using routers and switches while implementing **Static Routing** for communication between different networks.

The objective of this task was to understand hierarchical enterprise network architecture and demonstrate the Core, Aggregation, Distribution, and Access layers in a real-world network design.

---

## Project Objectives

- Design an enterprise-level network.
- Configure Static Routing between multiple routers.
- Build a hierarchical network architecture.
- Connect wired and wireless devices.
- Verify end-to-end communication.
- Understand the responsibilities of each network layer.

---

## Network Architecture

The enterprise network consists of:

### Core Layer
- High-speed backbone routers.
- Responsible for forwarding traffic between different parts of the network.

### Aggregation Layer
- Aggregates traffic from distribution switches.
- Connects the distribution layer with the core layer.
- Performs route forwarding between network segments.

### Distribution Layer
- Cisco switches connecting multiple access networks.
- Manages traffic before forwarding it to the aggregation layer.

### Access Layer
- Provides connectivity for end-user devices.
- Includes wired and wireless clients.
